# SEO (Search Engine Optimization)

Analyze the SEO of Rubycon from a number of points of view:
1. Being easy to find on Google by searching its Query terms below
2. If you find aggregators sites, see if Rubycon is there, and if not designate a path forward for our site to appear there. Could be: send email to site owner with "please add Rubycon to your site" or could be the aggregator aggregates based on some rules, so we want to oblige to that rule.
3. Feel free to do some tests and create a solution plan.
